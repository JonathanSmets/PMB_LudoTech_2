Sets pour les connecteurs sortants

Les sets pour les connecteurs sortants permettent de répartir les enregistrements proposés dans les connecteurs par sous ensembles thématiques.

Zoom

L'ajout d'un set se fait en cliquant sur le bouton Ajouter un set, la modification en cliquant sur le nom du set à modifier.

Zoom

Le set nécessite un nom du set , ainsi qu'un type (non modifiable ultérieurement).

Les types suivants sont possibles :

  • paniers de notices

  • recherche multi-critères de notices

  • paniers d'exemplaires

  • paniers d'emprunteurs (lecteurs)

Après enregistrement, il est nécessaire de revenir sur les propriétés du set afin d'en définir le contenu.

Set composé de paniers
Zoom

A ce niveau, il est possible de sélectionner les paniers qui vont alimenter le set ou d'inclure toutes les notices de la base dans le set (sauf pour les paniers de lecteurs).

La durée de vie du cache permet de stocker les enregistrements du set dans un cache, ce qui réduit le processus d'interrogation et limite la charge du système.

Set construit à partir d'une recherche multi-critères
Zoom

Cette recherche multi-critères peut être copiée depuis une équation DSI. Elle peut être également créée ou modifiée en cliquant sur le bouton Modifier cette recherche :

Zoom

Après affichage des résultats de la recherche multi-critères, il faut alors cliquer sur le bouton Associer à un set pour connecteur externe.

Zoom

Ne pas oublier de définir la durée de vie du cache.

Mise à jour manuelle
Zoom

Lors du retour sur la liste des sets, le bouton Mettre à jour manuellement effectue le remplissage du set.
