Paramétrer l'affichage des vignettes

Affichage des couvertures en gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Configuration de l'affichage en gestion

Paramètre

Valeur par défaut

Explication

book_pics_show

0

Affichage des couvertures de livres en gestion ?

0 : Non.

1 : Oui.

book_pics_url

URL des vignettes des notices, dans le chemin fourni, !!isbn!! sera remplacé par le code ISBN ou EAN de la notice purgé de tous les tirets ou points.

exemple : http://www.monsite/opac/images/vignettes/!!isbn!!.jpg

Affichage des couvertures en Opac

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC sous-zone Affichage des notices :

Configuration de l'affichage en Opac

Paramètre

Valeur par défaut

Explication

book_pics_url

URL des vignettes des notices, dans le chemin fourni, !!isbn!! sera remplacé par le code ISBN ou EAN de la notice purgé de tous les tirets ou points.

exemple : http://www.monsite/opac/images/vignettes/!!isbn!!.jpg

show_book_pics

0

Afficher les vignettes de livres dans les fiches ouvrages ?

0 : Non.

1 : Oui.

Vignettes des documents numériques

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Configuration de l'affichage des vignettes de documents numériques

Paramètre

Valeur par défaut

Explication

vignette_imagemagick

Chemin de l'exécutable ImageMagick (/usr/bin/imagemagick par exemple).

vignette_x

100

Largeur de la vignette créée pour un exemplaire numérique image.

vignette_y

100

Hauteur de la vignette créée pour un exemplaire numérique image.
