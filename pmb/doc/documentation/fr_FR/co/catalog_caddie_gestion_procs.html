Gestion des procédures

Zoom

Catalogue Paniers : Gestion Gestion des procédures

Ce menu permet de visualiser toutes les procédures disponibles et d'en rajouter selon les besoins.

Les procédures sont des requêtes SQL qui permettent d'aller chercher des documents, de les pointer ou bien d'agir sur le contenu d'un panier.

Modifier une procédure

Pour modifier une procédure cliquez sur son nom dans la liste des procédures.

Zoom

  • Type de la procédure : sélection / pointage ou action

  • Nom de la procédure : identifie la procédure

  • Code MySQL : code SQL de la procédure

  • Commentaire : indication supplémentaire sur la procédure (utilisation, cas particulier, etc.)

  • Autorisations : utilisateurs pouvant voir et utiliser cette procédure

Après modification, cliquez sur Enregistrer.

Pour supprimer la procédure, cliquez sur Supprimer.

Ajouter une procédure

Pour ajouter une procédure vous avez deux possibilités :

  • importer une procédure déjà existante

  • rédiger le code de la procédure vous-même

Pour importer une procédure, cliquez sur le bouton Importer.

Avec le bouton Parcourir rechercher la procédure sur le disque dur de votre ordinateur et cliquer sur Importer.

Pour ajouter une procédure vous même, cliquez sur Ajouter une procédure. Saisissez le code dans le champ à l'écran.

Pour ajouter une procédure, il est nécessaire de connaître le langage SQL ou de recopier un code préexistant.

Exporter une procédure

Le bouton Exporter vous permet d'exporter une procédure. Vous pouvez ensuite enregistrer le fichier obtenu sur votre disque dur, l'envoyer par mail, etc.

Tester une procédure

Le bouton Tester n'est utilisable que pour les procédures de collecte / pointage. Il vous permet de tester votre procédure pour en vérifier le fonctionnement.
