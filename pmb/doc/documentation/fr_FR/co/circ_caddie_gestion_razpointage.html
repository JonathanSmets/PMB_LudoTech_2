Effacer les pointages

Comme son nom l'indique, cette opération permet de remettre à zéro les pointages effectués dans un panier.

Choisir le panier pour lequel vous souhaitez effacer les pointages.

Zoom

Tous les éléments sont désormais non-pointés.
