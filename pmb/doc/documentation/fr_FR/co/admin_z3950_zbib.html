Serveurs

Les serveurs Z39.50 définis ici sont accessibles pour la recherche Z39.50 dans l'onglet Catalogue.

Zoom

L'ajout d'un serveur se fait en cliquant sur le bouton Ajouter, la modification en cliquant dans la liste sur le serveur à modifier.

Zoom

Chaque serveur Z39.50 possède un profil reprenant l'ensemble des paramètres nécessaires à sa définition.

Contactez directement l'administrateur du serveur Z39.50 pour obtenir ce profil. Vous pouvez également faire une recherche portant sur les termes "profil Z39.50" ou "serveur Z39.50" sur Google.

Si vous faites partie du réseau d'une BDP[1], vous pouvez contacter votre BDP et lui demander le profil de son serveur Z39.50.

PMB est livré avec les paramètres de différents serveurs correspondant à des besoins francophones et à la bibliothèque du Congrès américain.

Le paramétrage d'un serveur Z39.50 est composé de différents éléments :

  • Nom :

    c'est le nom du serveur, qui sera affiché à l'écran et que vous pouvez définir à votre convenance.

  • Utilisation :

    CATALOG (catalogage). PMB permet uniquement l'utilisation du Z39.50 pour le catalogage.

  • Base :

    nom de la base de données (information contenue dans le profil Z39.50 du serveur).

  • URL et Numéro de port :

    adresse du serveur et numéro de port à interroger (information contenue dans le profil Z39.50 du serveur).

  • Format :

    Format des données (Information contenue dans le profil Z39.50 du serveur). Certains serveurs proposent plusieurs formats.

  • langue du format SUTRS :

    (Information contenue dans le profil Z39.50 du serveur).

  • Utilisateur et Mot de passe :

    identifiants de connexion au serveur (Information contenue dans le profil Z39.50 du serveur)

  • Fichier script de paramétrage de l'import Z39.50 :

    permet de préciser, en fonction du serveur, un script de traitement de l'import différent de celui défini par défaut.

Après avoir renseigné ces différents éléments, cliquez sur le bouton Enregistrer.

Attributs

Le profil Z39.50 du serveur précise également différents attributs qui définissent les critères de recherche proposés par le serveur Z39.50.

Lors de la création du serveur, des attributs par défaut sont définis. Revenez en modification sur le serveur et cliquez sur le bouton Attributs pour les redéfinir.

Zoom

L'ajout d'un attribut se fait en cliquant sur le bouton Ajouter, la modification en cliquant dans la liste sur l'attribut à modifier.

Zoom

Chaque attribut se compose d'un Libellé à choisir dans une liste et d'un Attribut numérique.

N'oubliez pas de vérifier que le protocole Z39.50, ainsi que les URL et numéros de port définis pour chaque serveur sont bien autorisés dans votre environnement informatique.
