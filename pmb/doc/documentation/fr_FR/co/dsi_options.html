Gestion des classements

La création de plusieurs DSI engendre automatiquement un grand nombre de bannettes et d'équations qu'il peut être alors difficile d'utiliser.

Les classements autorisent une répartition des bannettes et équations permettant d'en simplifier la gestion.

Créer un nouveau classement

Dans Options > Classement, cliquez sur Ajouter un classement.

Indiquez un nom puis le type du classement (bannettes ou d'équations) dans le formulaire :

Zoom

Cliquez sur Enregistrer pour sauvegarder le classement. PMB affiche ensuite la liste des classements, dans laquelle vous retrouvez le classement nouvellement créé.

Zoom

Pour modifier ou supprimer un classement, cliquez sur le classement correspondant dans la liste.

La création des classements de bannettes et d'équations s'effectue sur la même interface. C'est le type de classement qui détermine sur quel élément portera le filtre.

Affecter un classement à une bannette ou à une équation

L'affectation d'un classement à une bannette/équation est déterminée dans ses propriétés.

Dans chaque équation, vous retrouvez les classements dans le menu déroulant :

Zoom

Pour les bannettes, le choix du classement s'effectue ici :

Zoom
Rechercher avec les filtres de classement

L'utilisation des classements s'effectue à l'affichage de toutes les bannettes publiques (Bannettes : Publiques) et de toutes les équations (Équations : Gestion).

Sélection des bannettes classées en "Secteur Jeunes" :
Zoom
Sélection des équations classées en "adulte".
Zoom
