Les tags

Vous pouvez autoriser les lecteurs à déposer un tag (mot-clé) sur la notice d'un document.

Les lecteurs peuvent ainsi participer à l'indexation des documents en proposant des mots-clés, associés à la notice du document.

La procédure s'effectue en deux temps :

  • Dans l'OPAC, le lecteur recherche la notice du document et y dépose un tag.

  • Les utilisateurs de PMB sont ensuite alertés en partie gestion. Ils traitent alors le tag du lecteur en le validant ou en le rejetant.

    Si le tag est validé, celui-ci s'affiche alors sur la notice du document, visible par les lecteurs.

Utilisation dans un centre de documentation spécialisé

Un chercheur pourra proposer une indexation complémentaire sur des documents techniquement complexes.

Paramétrage

Activation

La gestion des tags n'est pas activée par défaut dans PMB.

Pour plus d'informations, voir le Guide de l'administrateur PMB.
  • Paramétrer les tags

Proposer un nouveau tag

Comment déposer un tag sur une notice

Dans l'OPAC, accéder à la notice du document

Le lecteur recherche le document sur lequel il souhaite proposer un mot-clé.

Ajouter un nouveau tag

Le lecteur clique sur l'icône correspondante : .

Dans le pop-up qui s'affiche, deux choix sont proposés au lecteur :

  • saisie d'un tag à l'aide du menu déroulant : permet de sélectionner un mot-clé parmi ceux qui existent déjà dans la base.

    Si le lecteur choisit un tag dans la liste Choisissez un tag existant, le tag s'affichera automatiquement dans le champ libre en-dessous.

  • saisie d'un tag directement dans le champ libre Tapez en un nouveau : le lecteur opte pour cette solution si le tag qu'il souhaite proposer n'existe pas dans la liste.

Zoom

Le lecteur valide sa proposition de tag en cliquant sur Ajouter.

Le mot-clé ne s'affiche pas immédiatement sur la notice. Pour le voir, le lecteur devra attendre la validation de sa proposition.

Gestion des tags

Comment traiter un tag ?

En partie gestion, visualiser l'alerte "Tags à valider"

Après qu'un lecteur ait proposé un tag sur la notice d'un document, une alerte s'affiche dans l'onglet Catalogue :

L'utilisateur clique sur l'alerte pour afficher le (ou les) tag(s) à traiter.

Zoom

En cliquant sur l'alerte, vous accédez à Catalogue > Document : Gestion des tags

Traiter le tag

Traiter un tag se résume à l'accepter ou non.

Pour traiter un trag :

  • sélectionner le tag en cochant la case qui se trouve à sa droite (si l'on souhaite traiter tous les tags en une seule fois, cliquer sur Sélectionner)

  • accepter le tag en cliquant sur Valider

Zoom

Après validation :

  • La proposition du lecteur s'efface des tags à valider (l'alerte Tags à valider disparait de l'onglet Catalogue ).

  • Le tag est affiché dans les mots-clés libres de la notice.
