Paramètres généraux liés aux prêts

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Configuration liée à la gestion des prêts

Paramètre

Valeur par défaut

Explication

confirm_retour

0

En retour de documents, le retour doit-il être confirmé ?

0 : Non, on peut passer les codes-barres les uns après les autres.

1 : Oui, il faut valider le retour après chaque code-barre.

hide_retdoc_loc_error

0

Masquer le message d'erreur en retour de prêt d'un document issu d'une autre localisation.

pret_adhesion_depassee

1

Prêts si adhésion dépassée :

0 : INTERDIT incontournable.

1 : POSSIBLE.

pret_aff_limitation

0

Activer la limitation de l'affichage de la liste des prêts dans la fiche lecteur ?

0 : Inactif.

1 : Limitation activée.

pret_aff_nombre

10

Nombre de prêts à afficher si le paramètre pret_aff_limitation est actif.

0 : tout voir, illimité.

N : Nombre de prêts à afficher sur la première page.

pret_date_retour_adhesion_depassee

0

La date de retour peut-elle dépasser la date de fin d'adhésion ?

0 : Non, la date de retour sera calculée pour ne pas dépasser la date de fin d'adhésion.

1 : Oui, la date de retour du prêt sera indépendante de date de fin d'adhésion.

pret_express_statut

2

Statut de notice à utiliser en création d'exemplaires en prêt express.

selfservice_allow

0

Active la gestion de la borne de prêt?

0 : Non.

1 : Oui.

Prolongations des prêts

En gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Prolongations des prêts

Paramètre

Valeur par défaut

Explication

pret_nombre_prolongation

3

Nombre de prolongations autorisées.

pret_restriction_prolongation

0

Restreindre les prolongations de prêts ?

0 : pas de restriction.

1 : prolongation limitée au paramètre pret_nombre_prolongation.

2 : prolongation gérée par les quotas.

Pour plus d'informations sur les prolongations gérées par les quotas, voir le chapitre :
  • Quotas
En Opac

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Options générales de fonctionnement de l'OPAC :

Prolongations des prêts en OPAC par le lecteur

Paramètre

Valeur par défaut

Explication

pret_duree_prolongation

10

Nombre de jours de prolongation autorisé.

pret_prolongation

0

Autoriser la prolongation par le lecteur ?

0 : Non.

1 : Oui.

Imprimante ticket de prêt

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Imprimante ticket de prêt

Paramètre

Valeur par défaut

Explication

printer_ticket_script

Script permettant de personaliser l'impression du ticket de prêt.

Le répertoire du script est à paramétrer à partir de la racine de PMB.

Si vide PMB utilise ./circ/ticket-pret.inc.php

printer_ticket_url

Permet d'utiliser une imprimante de ticket, connectée en local sur le poste de prêt client.

Vide : pas d'imprimante.

Url (http://localhost/printer/bixolon_srp350.php ) : imprimante active.

Configurer l'archivage des prêts en gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Emprunteurs  :

Configuration de l'archivage des prêts

Paramètre

Valeur par défaut

Explication

archivage_prets

0

Archiver les prêts des emprunteurs ?

0 : Non.

1 : Oui.

ATTENTION pour la France: nous attirons votre attention sur l'obligation de déclarer votre traitement à la CNIL (www.cnil.fr) si vous activez cette fonctionnalité.

archivage_prets_purge

0

Nombre de jours maximum où doivent être conservées les archives nominatives de prêts :

0 : illimité.

N : N jours.

Configurer l'archivage des prêts en OPAC

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC , sous-zone Options générales de fonctionnement de l'OPAC :

Configuration de l'affichage de l'historique des prêts sous le compte lecteur

Paramètre

Valeur par défaut

Explication

empr_hist_nb_jour_max

0

Délai en jours au-delà duquel les prêts précédents ne sont pas affichés :

0 : pour illimité

N : N jours

empr_hist_nb_max

0

Nombre maximum de prêts précédents à afficher :

0 : pour illimité.

N : N prêts.

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC , sous-zone Modules affichés dans l'OPAC :

L'archivage des prêts, permet d'afficher sous l'opac les autres lectures.

Ex : les lecteurs qui ont lu cet ouvrage, ont également lu...

Configuration des autres lectures

Paramètre

Valeur par défaut

Explication

autres_lectures

0

Afficher les emprunts des autres lecteurs du document courant ?

0 : Non.

1 : Oui.

autres_lectures_nb_jours_maxi

0

Délai en jours au-delà duquel les emprunts ne sont pas comptabilisés :

0 : pour illimité.

N : N jours.

autres_lectures_nb_maxi

0

Nombre maximum de lectures associées proposées.

autres_lectures_nb_mini_emprunts

100

Nombre minimum d'emprunts pour être comptabilisés :

1: un seul emprunt suffit pour proposer la notice comme lecture associée.

N: N emprunts minimum nécessaires.

autres_lectures_tri

rand()

Tri des autres lectures proposées :

rand(): aléatoire.

tit: par Titre.
