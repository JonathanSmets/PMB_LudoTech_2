Procédures distantes

Les procédures distantes permettent de consulter un serveur qui contient des requêtes pour les paniers. Vous pouvez ensuite importer celles qui vous intéressent.

Pour plus d'informations, voir le Guide de l'administrateur PMB.
  • Configurer le serveur de procédures
Zoom

Le bouton Tester permet de vérifier la procédure distante, le bouton Importer de l'ajouter à la liste des procédures enregistrées dans le logiciel.
