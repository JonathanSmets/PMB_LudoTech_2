Sous-collections

Sous-collection

Une sous-collection est une subdivision thématique d'une collection.

Dans une collection (un thème), l'éditeur peut décider de créer des sous-collections (des sous thèmes).

Une collection Histoire pourra comprendre des sous-collections Préhistoire, Moyen Age, etc.

Recherche

L'écran de recherche des sous-collections est composé d'un formulaire de recherche et d'une liste de résultats.

Zoom

Le formulaire de recherche comporte :

  • une zone de saisie des critères de recherche (dans laquelle les expressions booléennes sont acceptées),

  • un lien Voir les dernières sous-collections créées.

La liste de résultats, paginée, précise également le nombre d'utilisations des sous-collections dans le catalogue.

Un clic sur le nombre d'utilisations renvoie la liste des notices pour lesquelles la sous-collection correspondante est référencée.

Création / modification

L'ajout d'une sous-collection se fait avec le bouton Créer une sous-collection, la modification en cliquant sur le nom de la sous-collection dans la liste.

Zoom

Une sous-collection est composée des informations suivantes :

  • Nom (obligatoire),

  • ISSN : identifiant unique normalisé de la sous-collection,

  • Sous-collection de : champ contrôlé (obligatoire),

    une collection est forcément liée à une sous-collection.

  • Éditeur (automatique)

  • Site web : ou page de référence sur la sous-collection.

    Le bouton Vérifier permet de vérifier si l'URL est toujours valide. Il est nécessaire de renseigner l'URL complète (ex : http://www.site_de_la_sous_collection.com).

  • Commentaire : informations complémentaires sur la sous-collection.

Le bouton Utilisation, accessible en modification, renvoie la liste des notices pour lesquelles la sous-collection correspondante est référencée.
