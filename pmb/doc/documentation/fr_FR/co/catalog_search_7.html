Introduction

Catalogue > Recherche : Toutes notices > Externe

La recherche externe est une recherche par connecteurs.

C'est à dire que vous allez rechercher des notices à l'extérieur du logiciel pour éventuellement les intégrer dans votre catalogue. Les connecteurs sont des sites internet, des entrepôts OAI, des catalogues d'autres bibliothèques, etc.

Les connecteurs doivent être préalablement paramétrés en onglet Administration.

Pour plus d'informations, voir le Guide de l'administrateur PMB.
  • Connecteurs entrants

Cette recherche est également accessible par Catalogue >  Externe : Connecteurs

Utilisation

Zoom

Pour utiliser cette recherche vous devez avoir paramétré au moins une source. La source représente l'accès à un catalogue en particulier.

Pour l'affichage les sources peuvent être regroupées par catégories, ici Amazon, Spécialisées et Généralistes. Ces catégories sont également paramétrables en onglet Administration.

Cochez au moins une source.

Les critères de recherches fonctionnent comme une recherche multi-critères. Choisissez un champ et remplissez le avant de cliquer sur Rechercher.

Ne sélectionnez pas trop de sources pour une recherche car cela alourdit le processus.

Intégrer une notice dans le catalogue

Les notices trouvées par une recherche externe sont intégrables dans le catalogue.

Recherche avancée

L'interrogation externe avancée vous permet d'utiliser une recherche multi-critères avec l'intégralité des champs disponibles dans PMB.

Si votre recherche porte sur un champ qui n'est pas géré par le connecteur, le critère est ignoré : ce sont tous les enregistrements qui sont retournés.

Comment réaliser une recherche externe avancée

Catalogue Externe  : Connecteurs

Cliquer sur Faire une recherche avancée

Choisir la ou les sources à utiliser

Pour choisir plusieurs sources utilisez la touche clavier Ctrl.

Définir les critères de la recherche

Zoom

L'ergonomie est la même que celle de la recherche multi-critères.

Pour plus d'informations, consulter le chapitre Recherche toutes notices : multi-critères dans ce même guide.
  • Recherche multi-critères

Cliquer sur Rechercher

Il n'y a pas de différences dans la présentation des résultats entre la recherche externe simple et la recherche externe avancée.
