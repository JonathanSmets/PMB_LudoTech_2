Imports

L'import permet d'intégrer dans le catalogue un fichier de notices bibliographiques, conforme au format UNIMARC[1].

Ce fichier peut être fourni par une BDP[2] qui dépose des ouvrages dans une bibliothèque de son réseau, ou provenir de différentes sources de données bibliographiques (éditeurs, fournisseurs, sites internet...).

Si l'import ne peut se faire qu'à partir d'un fichier conforme au format UNIMARC[1], PMB permet la conversion vers ce format d'autres types de fichiers.

Pour plus d'informations, voir le chapitre :
  • Conversions/export
Références :
  1. UNIMARC

    Basé sur le format MARC (MAchine-Readable Cataloging), UNIMARC en est une variante créé par l'IFLA en 1977. Ce format est le format officiel d'échange de données bibliographiques en France.

  2. BDP

    Bibliothèque Départementale de Prêt
