Visionneuse

Dans l'OPAC[1], le fait de cliquer sur une icône de document numérique provoque un envoi de ce document à destination du navigateur.

A réception, en fonction du MIMEtype[2] de celui-ci, le navigateur peut :

  • Ouvrir le document

  • Proposer son téléchargement, ou son affichage dans une application tierce.

  • Une image au format GIF sera par exemple ouverte dans le navigateur.

  • Un fichier au format PDF pourra être affiché par le navigateur si celui-ci possède une extension (plugin) le permettant, proposé au téléchargement ou affiché dans une application telle qu'Adobe Reader.

En ce qui concerne les applications, le comportement du navigateur peut être redéfini.

La Visionneuse propose un mécanisme de redéfinition de l'affichage des documents numériques dans l'OPAC[1][2].

Le paramétrage définit les MIMEtypes pour lesquels les documents sont ouverts dans la visionneuse. Cet affichage est réalisé dans la page (iframe[3]).

La visionneuse propose un ensemble de classes d'affichage qui prennent en charge certains MIME Types. Pour les autres, la visionneuse laisse le navigateur gérer l'affichage.

Actuellement, les classes d'affichage disponibles sont les suivantes :

  • images : supporte les MIMEtypes 'image/png', 'image/x-png','image/jpg','image/jpeg',image/gif',

  • pdf : supporte le MIMEType 'application/pdf'.

Références :
  1. OPAC

    Online Public Access Catalog. (Catalogue public en ligne)

  2. MIMEtype
    Multipurpose Internet Mail Extensions Type MIME Content-type

    Un MIMEtype est un système permettant d'identifier un format de données.

    Il est composé à minima d'un type et d'un sous-type (ex : application/pdf, image/gif).

  3. iframe

    Une iframe est une page HTML incorporée dans une autre page.
