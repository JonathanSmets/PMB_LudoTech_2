Les modes de choix des paramètres possibles sont :

  • Texte

    Une zone de texte libre affichée sur une ligne.

  • Liste de choix

    Une liste prédéfinie affichée sous la forme d'un sélecteur, dont les valeurs sont fixées dans les options du champ personnalisable.

  • Liste de choix à partir d'une requête

    Une liste prédéfinie affichée également sous la forme d'un sélecteur, et dont les valeurs sont déterminées à partir d'une requête MySQL.

  • Sélection d'une date

    Une date formatée, dont la saisie se fait avec un sélecteur de type calendrier.

  • Valeurs à partir d'un fichier

    Une zone de texte libre pouvant être affichée sur plusieurs lignes.

  • Valeurs à partir d'un sélecteur d'autorités

    Un sélecteur de données défini à partir d'un script externe au logiciel PMB.

Options des paramètres d'actions personnalisables

Pour chacun des modes de choix des paramètres d'actions personnalisables, il est nécessaire de préciser les options possibles en cliquant sur le bouton Options.

Options pour le mode de choix de type "Texte"

  • Taille du champ à l'écran

    permet de fixer la largeur de la zone d'affichage présentée à l'écran lors de la saisie.

  • Nombre maximum de caractères à saisir

    permet de limiter la taille de l'information saisie.

Options pour le mode de choix "Liste de choix"

  • Liste multiple

    permet de préciser un champ multivalué.

  • Valeur et Choix vide

    Valeur et libellé prédéfinis si aucun choix n'est effectué lors de la saisie du champ personnalisable.

  • Liste des options de la liste

    Elle se compose :

    • d'une Valeur, stockée en base.

    • d'un Libellé, affiché pour la saisie.

Options pour le mode de choix "Liste de choix à partir d'une requête"

  • Liste multiple

    permet de préciser un champ multivalué.

  • Valeur et Choix vide

    Valeur et libellé prédéfinis si aucun choix n'est effectué lors de la saisie du champ personnalisable.

  • Requête

    Cette requête interroge une table appartenant à la base. Elle ne doit retourner qu'un seul champ.

Le bouton Tester permet de vérifier la requête MySQL.

Sélection d'une date

Aucune option n'est nécessaire pour la sélection d'une date. Il faut néanmoins cliquer sur le bouton Options et valider le message "Les options par défaut de la boite de sélection de date ont été configurées !"

Options pour le mode de choix "Valeurs à partir d'un fichier"

Le fichier est un fichier texte et doit être composé d'une valeur par ligne.

  • Méthode de récupération des données

    • Liste récupère les valeurs possibles directement depuis le fichier.

    • Table temporaire stocke les valeurs possibles dans une table temporaire avant de les récupérer.

  • Nom de la table temporaire si nécessaire

    voir ci dessus.

  • Type de données

    précise si les valeurs doivent être considérées en tant que chaînes de caractères ou en tant qu'entiers.

Options pour le mode de choix "Liste de choix à partir d'un sélecteur d'autorités"

  • Méthode de récupération des valeurs

    permet de préciser si l'on récupère l'identifiant (Id) de l'autorité ou son Libellé (label).

  • Type de données

    les autorités qu'il est possible d'interroger :

    • Auteurs

    • Catégories

    • Editeurs

    • Collections

    • Sous-collections

    • Titres de séries

    • Indexations décimales
