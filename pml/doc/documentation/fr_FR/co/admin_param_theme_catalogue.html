Paramètres liés aux périodiques

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés à la gestion des périodiques

Paramètre

Valeur par défaut

Explication

abt_end_delay

30

Délais d'alerte d'avertissement des abonnements arrivant à échéance (en jours)

first_week_day_format

0

Format de la semaine pour les abonnements :

0 : la semaine commence le lundi.

1 :l a semaine commence le dimanche.

serial_link_article

0

Pré-remplissage du lien des dépouillements avec le lien de la notice mère en catalogage des périodiques ?

0 : Non.

1 : Oui.

Paramètres liés à la gestion des paniers

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés à la gestion des paniers

Paramètre

Valeur par défaut

Explication

confirm_delete_from_caddie

0

Action à réaliser lors de la suppression d'une notice située dans un panier :

0 : Interdire ;

1 : Supprimer sans confirmation.

2 : Demander une confirmation de suppression.

Paramètres liés à la recherche et au tri

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés à la recherche et au tri

Paramètre

Valeur par défaut

Explication

default_operator

0

Opérateur par défaut :

0 : OR.

1 : AND.

extended_search_auto

1

En recherche multi-critères, la sélection d'un champ ajoute celui-ci automatiquement sans avoir besoin de cliquer sur le bouton Ajouter ?

0 : Non.

1 : Oui.

multi_search_operator

or

Type d'opérateur de recherche pour les listes avec plusieurs valeurs:

or : pour le OU.

and : pour le ET.

nb_max_tri

50

Nombre maximum de notices pour lesquelles le tri est autorisé.

recherche_ajax_mode

1

Affichage accéléré des résultats de recherche: "réduit" uniquement, la suite est chargée lors du click sur le "+".

0 : Inactif.

1 : Actif.

title_ponderation

0.5

Majoration de la pondération des mots du titre.

Mettre 0 (zero) pour interdire la majoration.

ATTENTION utiliser le point décimal

Paramètres liés aux notices

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés aux notices

Paramètre

Valeur par défaut

Explication

form_editables

0

Grilles de notices éditables ?

0 : Non.

1 : Oui.

Keyword_sep

;

Séparateur des mots clés dans la partie indexation libre, espace ou ; ou , ou ...

latest_order

notice_id desc

Tri des dernières notices ?

notice_id desc : par id de notice décroissant: idéal mais peut être problématique après une migration ou un import.

create_date desc: par la colonne date de création.

nb_lastnotices

10

Nombre de dernières notices affichées en catalogue (dernières notices).

notice_controle_doublons

Contrôle sur les doublons en saisie de la notice :

0 : Pas de contrôle sur les doublons.

1,tit1,tit2, ... : Recherche par méthode _exacte_ de doublons sur des champs, défini dans le fichier notice.xml

2,tit1,tit2, ... : Recherche par _similitude_

notice_fille_format

0

Affichage des notices filles :

0 : avec leurs détails (notice dépliable avec un plus).

1 : Juste l'entête.

notice_reduit_format

0

Format d'affichage des réduits des notices :

0 : titre+auteur principal.

1 : titre+auteur principal+date édition.

pret_express_statut

2

Statut de notice à utiliser en création d'exemplaires en prêt express.

show_notice_id

0

Afficher l'identifiant de la notice dans le descriptif ?

0 : Non.

1 : Oui.

1,X : Oui avec préfixe X.

type_audit

0

Gestion/affichage des dates de création/modification ?

0 : Rien.

1 : Création et dernière modification.

2 : Création et toutes les dates de modification.

use_uniform_title

0

Utiliser les titres uniformes ?

0 : Non.

1 : Oui.

Paramètres liés aux exemplaires

Onglet Administration, menu Outils, sous-menu Paramètres, zone Paramètres généraux  :

Paramètres généraux liés aux exemplaires

Paramètre

Valeur par défaut

Explication

expl_list_display_comments

0

Afficher les commentaires des exemplaires en liste d'exemplaires :

0 : non.

1 : commentaire bloquant.

2 : commentaire non bloquant.

3 : les deux commentaires.

expl_show_dates

0

Afficher les dates de création et de modification des exemplaires ?

0 : Non.

1 : Oui.

expl_title_display_format

expl_location,expl_section,expl_cote,expl_cb

Format d'affichage du titre de l'exemplaire en recherche multi-critères d'exemplaires.

Les libellés des champs correspondent aux champs de la table exemplaires, ou aux id de champs personnalisés. Séparés par une virgule. Les champs disposant d'un libellé seront remplacés par le libellé correspondant.

Exemple: expl_location,expl_section,expl_cote,expl_cb

label_construct_script

Script de construction d'étiquette de cote

numero_exemplaire_auto

0

Autorise la numérotation automatique d'exemplaire ?

0 : Non.

1 : Oui.

numero_exemplaire_auto_script

gen_code/gen_code_exemplaire.php

Nom du fichier de script php pour la génération des codes d'exemplaires en automatique.

prefill_cote

custom_cote_02.inc.php

Script personnalisé de construction de la cote de l'exemplaire.

prefill_cote_ajax

Script personnalisé de construction de la cote de l'exemplaire en ajax.

type_audit

0

Gestion/affichage des dates de création/modification ?

0 : Rien.

1 : Création et dernière modification.

2 : Création et toutes les dates de modification.
