Activer et configurer la gestion multi-thésaurus en partie Gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Catégories, thésaurus et plans de classements, sous-zone Options pour les thésaurus :

Activation et configuration de la gestion multi-thésaurus en partie Gestion

Paramètre

Valeur par défaut

Explication

mode_pmb

0

Niveau d'utilisation des thésaurus.

0 : Un seul thésaurus par défaut.

1 : Choix du thésaurus possible.

defaut

0

Identifiant du thésaurus par défaut.

liste_trad

fr_FR

Liste des langues affichées dans les thésaurus.

(ex : fr_FR,en_UK,...,ar)

Options d'affichage des catégories en partie Gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Catégories, thésaurus et plans de classements, sous-zone Options pour les categories :

Options d'affichage des catégories en partie Gestion

Paramètre

Valeur par défaut

Explication

categories_affichage_ordre

0

Paramétrage de l'ordre d'affichage des catégories d'une notice.

0 : Par ordre alphabétique

1 : Par ordre de saisie

categories_categ_in_line

0

Affichage des catégories en ligne.

0 : Non

1 : Oui

categories_show_only_last

0

Dans la fiche d'une notice :

0 : Tout afficher

1 : Afficher uniquement la dernière feuille de l'arbre de la catégorie

categories_term_search_n_per_page

50

Nombre de termes affichés par page lors d'une recherche par terme dans les catégories

Autopostage en partie Gestion

Onglet Administration, menu Outils, sous-menu Paramètres, zone Catégories, thésaurus et plans de classements, sous-zone Options pour l'affichage et la navigation dans les catégories :

Configuration de l'autopostage en partie Gestion

Paramètre

Valeur par défaut

Explication

auto_postage_descendant

0

Activer la recherche des notices des catégories filles.

0 : Non

1 : Oui

auto_postage_nb_descendant

0

Nombre de niveaux de recherche de notices dans les catégories filles.

* : Illimité.

N : Nombre de niveaux.

auto_postage_montant

0

Activer la recherche des notices des catégories mères ?

0 : Non

1 : Oui

auto_postage_nb_montant

0

Nombre de niveaux de recherche de notices dans les catégories mères.

* : Illimité.

N : Nombre de niveaux.

auto_postage_etendre_recherche

0

Proposer la possibilité d'étendre la recherche dans les catégories mères ou filles.

0 : Non

1 : Exclusivement dans les catégories filles.

2 : Etendre dans les catégories mères et filles.

3 : Exclusivement dans les catégories mères.

Configurer le navigateur de catégories en OPAC

Configuration du navigateur de catégories en OPAC

Paramètre

Valeur par défaut

Explication

show_categ_browser

0

Affichage des catégories en page d'accueil OPAC :

0 : Non.

1 : Oui.

Onglet Administration, menu Outils, sous-menu Paramètres, zone OPAC, sous-zone Options pour l'affichage et la navigation dans les catégories :

Configuration du navigateur de catégories en OPAC

Paramètre

Valeur par défaut

Explication

categories_columns

3

Nombre de colonnes du sommaire général des catégories.

categories_max_display

200

Pour la page d'accueil, nombre maximum de catégories principales affichées.

categories_nav_max_display

200

Limiter l'affichage des catégories en navigation dans les sous-catégories.

0 : Pas de limitation.

>0 : Nombre max de catégories à afficher.

categories_sub_display

3

Nombre de sous-categories sur la première page.

categories_sub_mode

libelle_categorie

Mode affichage des sous-categories :

rand() : aléatoire.

libelle_categorie  : ordre alpha.

categories_nb_col_subcat

3

Nombre de colonnes de sous-catégories en navigation dans les catégories.

categories_categ_rec_per_page

6

Nombre de notices à afficher par page dans l'exploration des catégories.

categories_categ_sort_records

index_serie, tnvol, index_sew

Explorateur de catégories, mode de tri des notices :

index_serie, tnvol, index_sew : par titre de série, numéro dans la série et index des titres.

rand() : aléatoire
