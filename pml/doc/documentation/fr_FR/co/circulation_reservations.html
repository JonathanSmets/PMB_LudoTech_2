Réservations

PMB permet de réserver des documents pour un lecteur.

Les réservations s'effectuent à plusieurs endroits :

  • Dans l'onglet Circulation, sur chaque fiche lecteur (par l'intermédiaire du menu Prêt de document).

  • Dans l'onglet Catalogue, sur chaque notice de document.

  • Suivant le paramétrage, il est également possible pour un lecteur de réserver un document dans l'OPAC (après s'être identifié).

Pour plus d'informations, voir le Guide de l'administrateur PMB.
