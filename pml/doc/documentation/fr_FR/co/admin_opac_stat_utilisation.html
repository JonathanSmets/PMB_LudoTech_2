Créer une statistique OPAC.

Pour créer une statistique OPAC, il faut donc définir :

1. Une vue.

2. des colonnes.

3. une requête.

Créer une vue.
Zoom

L'ajout d'une vue se fait en cliquant sur le bouton Ajouter une vue.

Zoom

Chaque vue nécessite un Nom de la vue et éventuellement une Description de la vue.

Créer une ou plusieurs colonnes
Zoom

Après enregistrement de la vue, pour atteindre la grille de saisie d'une colonne il faut cliquer sur le nom de la vue.

Zoom

L'ajout d'une colonne se fait en cliquant sur le bouton Ajouter une colonne.

Zoom

  • Nom de la colonne :

    Ce nom est le nom stocké dans la base de données. Il ne peut comporter que des caractères alphanumériques non accentués, ainsi que le caractère "_" (underscore).

  • Expression associée :

    fonction permettant de récupérer les données utiles (cf. liste des fonctions disponibles pour les colonnes).

  • Filtre associé :

    Il permet de filtrer la colonne à partir d'une autre fonction.

  • Type de données :

    correspondent aux formats utilisés pour le stockage des données.

    Ces formats sont liés aux expressions associées.

Zoom

Pour modifier une colonne, cliquez sur la ligne du tableau correspondant à la ligne à modifier.

Le bouton Réinitialiser la vue supprime l'ensemble des colonnes et requêtes de la vue.

Créer une requête
Zoom

Après enregistrement de la vue, pour atteindre la grille de saisie d'une colonne, dépliez la vue concernée en cliquant sur l'icône puis sur le bouton Ajouter une requête.

Zoom

  • donnez un Nom à votre procédure

  • saisissez le Code MySQL de la requête.

    Les colonnes disponibles peuvent être insérées avec le bouton << à partir d'une sélection dans la zone Colonnes associées.

  • Ajoutez éventuellement un Commentaire.

Zoom

Après enregistrement de la requête, celle -ci peut être modifiée. Dépliez la vue concernée en cliquant sur l'icône puis cliquez sur le nom de la requête.

Consolider les vues

La consolidation permet de mettre à jour les données collectées pour les statistiques.

Dans les options de consolidation, sélectionnez l'option souhaitée et lancez l'opération.

Zoom

Cochez les vues à consolider, dépliez les options de consolidation en cliquant sur l'icône , sélectionnez l'option choisie puis cliquez sur le bouton Consolider les vues.

La consolidation des vues est un processus coûteux en ressources serveur, il est donc préférable d'exécuter cette opération dans des tranches horaires de faible utilisation du serveur.

Consulter les statistiques

Après l'élaboration des statistiques en administration, celles-ci sont consultables dans l'onglet Éditions, menu Opac, sous-menu Statistiques de l'Opac.

Pour exécuter la statistique souhaitée, vous devez cliquer sur le nom de la statistique.

À cette étape, vous pouvez exporter le résultat vers un tableur en cliquant sur une des icônes situées en bas de page.

Pour plus d'informations, voir le chapitre :
  • Statistiques de l'OPAC
