Importer / Exporter une action personnalisable

Import / export d'actions personnalisables sous la forme de fichiers

Zoom

Dans la liste des actions personnalisables, le bouton Exporter permet d'enregistrer la procédure concernée sous forme de fichier.

En bas de liste, le bouton Importer permet d'effectuer l'opération inverse.

Si la société PMB Services propose la réalisation de procédures dans le cadre de ses prestations, le Wiki, la mailing-liste des utilisateurs et bien d'autres sources proposent des actions personnalisables. (Voir notamment l'espace "PMB La communauté" sur le site de PMB Services.

Import de procédures distantes

PMB Services propose dans le cadre de ses prestations un serveur de procédures.

Ce serveur permet de d'exécuter et d'importer des procédures génériques.
