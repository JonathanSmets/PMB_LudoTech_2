Dictionnaire des synonymes

Synonyme

C'est un mot de forme différente mais de même sens qu'un autre mot. Il peut se substituer à un autre dans un même contexte. L'utilisateur de PMB crée son dictionnaire de synonymes en fonction de ses besoins.

Zoom

Vous pouvez enregistrer dans PMB vos synonymes dans un dictionnaire. Lors des recherches, un mot présent dans le dictionnaire des synonymes avec un renvoi sera remplacé par un autre avant que la recherche ne soit effectuée.

Le dictionnaire va servir pour des sigles, des pluriels, des pluriels irréguliers, des mots de domaines spécialisés... Les inscrire dans ce dictionnaire va permettre de faire des renvois entre les mots.

Ajouter des mots et créer un dictionnaire de synonymes

Dans la page de gestion des synonymes vous pouvez cliquer sur Créer un mot et remplir le formulaire.

Mot : mot,sigle ou expression porteurs de sens. Saisissez le mot à ajouter au dictionnaire

Synonymes  : avec le bouton ... allez chercher un mot du dictionnaire synonyme du précédent.

Rechercher un mot dans le dictionnaire

Pour rechercher un mot dans le dictionnaire vous pouvez utiliser:

  • le lien Voir les derniers mots créés.

  • le champ de recherche.

  • l' abécédaire des synonymes.

Cliquez sur une lettre pour visualiser les mots commençant par celle ci.

Cliquez sur le mot pour le modifier.

Vélo et Bicyclette

Cliquez sur Créer un mot.

Remplissez le formulaire pour le mot "Vélo".

Cliquez sur Créer un mot.

Remplissez le formulaire pour le mot "Bicyclette"

La réciprocité n'est pas automatique. vous pouvez rechercher le mot "Vélo" pour lui donner comme synonyme le mot "Bicyclette".
