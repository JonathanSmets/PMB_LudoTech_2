Actions

Communication

Lorsqu'une action du type Question/Réponse est créée soit par le lecteur demandeur soit par l'utilisateur (avec pour statut Ouvert), elle apparaît dans les fils de discussion.

Consulter les fils de discussion ouverts

Aller dans Demandes > Actions : Communication

Toutes les actions Question/Réponse dont le statut est en cours sont affichées dans Demandes > Actions : Communication.

Pour afficher l'action Question/Réponse et pour pouvoir y ajouter une note, vous pouvez cliquer sur une action dans la liste.

Demandes > Actions : Communication et cliquer sur une action Question/Réponse en cours revient à aller dans Demandes > Listes : Demandes validées et cliquer sur une demande puis cliquer sur une action question/réponse dans la liste des actions.

Planning des rendez-vous

Les rendez-vous affichés dans le planning des rendez-vous correspondent aux actions de type Rendez-vous dont le statut est "en cours".

Rendez-vous en attente

Les actions de type Rendez-vous sont consultables directement ici. Selon leur statut, elles sont visibles dans Demandes > Actions : Rendez-vous en attente ou Demandes > Actions : Rendez-vous en cours.

Consulter les rendez-vous pris ou planifiés

Pour consulter les rendez-vous en attente, aller dans Demandes > Actions : Rendez-vous en attente

Pour consulter les rendez-vous en cours, aller dans Demandes > Actions : Rendez-vous en cours

Pour passer un rendez-vous en attente en rendez-vous en cours, il faut modifier l'action correspondante et changer son statut.
