Conversion de fichiers externes

Les conversions permettent de transformer un fichier contenant des notices bibliographiques d'un format dans un autre.

Zoom

Le fichier à convertir peut être transmis de deux façons :

  • par téléchargement depuis l'interface du logiciel en cliquant sur le bouton Parcourir.

  • en déposant le fichier sur le serveur dans le répertoire pmb/temp (avec un client FTP par exemple).

Vous devez ensuite choisir un Type de conversion , puis cliquer sur le bouton Lancer la conversion.

Pour plus d'informations, voir le chapitre :
  • Formats d'export et de conversion

Une fois la conversion lancée, une barre de progression indique l'avancement du processus de conversion.

Finalement, le fichier généré par la conversion est proposé au téléchargement.

Zoom

Cliquez sur le bouton Terminer l'étape de conversion afin de le récupérer.

Dans le cas d'une conversion vers un format UNIMARC ISO2079, il est possible de lancer directement l'import du fichier généré.

Zoom
Pour plus d'informations, voir le chapitre :
  • Imports
