Visualiser

Le menu Visualiser permet d'afficher des informations sur vos documents.

Afin de faire une recherche documentaire, cette fonctionnalité est particulièrement utile pour les utilisateurs n'ayant pas accès à l'onglet Catalogue.

Raccourci clavier

Depuis n'importe quelle page, il est possible d'accéder directement à la recherche exemplaire en utilisant un raccourci clavier : appuyer sur la touche Echap de votre clavier, puis sur la touche V.
