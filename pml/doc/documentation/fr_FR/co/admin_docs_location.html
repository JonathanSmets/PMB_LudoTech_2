Localisations

Les localisations permettent d'indiquer l'emplacement des exemplaires.

En fonction du contexte, multi-sites ou mono-site, les localisations correspondent aux différents sites ou locaux dans lesquels sont stockés les exemplaires.

Zoom

L'ajout d'une localisation se fait en cliquant sur le bouton Ajouter une localisation, la modification en cliquant dans la liste sur la localisation à modifier.

Zoom

Pour chaque localisation, il est possible de spécifier :

  • une image pour affichage dans la navigation par localisation/section dans l'OPAC[1].

    (image disponible dans un des sous répertoires du répertoire opac_css de l'installation)

  • la visibilité de la localisation dans l'OPAC[1]

  • une infopage dans l'OPAC[1].

Pour plus d'informations, voir le chapitre :
  • Infopages

  • une adresse, un logo et un mail utilisés notamment pour les relances.

  • un commentaire visible en OPAC[1].

Les informations propriétaire du codage et codage d'import sont utilisées et mises à jour lors de l'import d'exemplaires.

Elles permettent de faire le lien entre les informations des exemplaires importés et le paramétrage des localisations spécifié dans le logiciel.

Pour plus d'informations, voir le chapitre :
  • Imports

Si les réservations localisées sont activées, il est possible de spécifier, en fonction de leurs localisations respectives, quels sont les exemplaires qui peuvent être réservés par les lecteurs.
