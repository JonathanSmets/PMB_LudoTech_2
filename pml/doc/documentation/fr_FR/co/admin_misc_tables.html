Description des tables

La description des tables récapitule la structure des tables de données de la base MySQL.

Zoom

Pour faire apparaître / disparaître le détail des tables, cliquez sur les liens.

Une documentation approfondie, comportant le contenu de chaque champ de la base de données ainsi que les relations entre les tables, est disponible à l'adresse http://SERVEUR/PMB/doc/noyau

(Les parties "SERVEUR" et "PMB" de cette adresse doivent être adaptées à votre configuration)
