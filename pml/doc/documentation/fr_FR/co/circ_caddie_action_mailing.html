Mailing

Le Mailing permet d'envoyer un mail à l'ensemble des lecteurs contenus dans un panier.

Le mail peut être écrit dans un format HTML, qui permet une mise en forme (présentation).

Dans ce cas, l'utilisation d'un éditeur HTML est possible.

Pour plus d'informations, voir le Guide de l'administrateur PMB.
  • Paramétrer l'éditeur javascript HTML

  • Cliquer sur le panier contenant la liste des lecteurs destinataires.

Zoom

  • Saisir l'objet et le corps du mail.

Zoom

  • Cliquer sur Envoyer.

Remise à zéro des envois

Lors du mailing, les lecteurs pour lesquels l'envoi s'est correctement déroulé sont pointés.

Le lien Remise à zéro des envois permet d'effacer ce pointage.
